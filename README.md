# RestAPI, Flask

Credit scoring model with Flask.

Setup Instructions
1. Clone the repository:
    ```bash
    git clone git@github.com/cptcroissant/OTUS-RestAPI.git
    cd OTUS-RestAPI
    ```
2. Create a project:
    ```bash
    uv python pin 3.11
    uv init
    uv venv
    uv add --requirements requirements.txt
   ```
